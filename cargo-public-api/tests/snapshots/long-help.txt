Usage: cargo public-api [--manifest-path <manifest-path>] [-p <package>] [--omit <omit...>] [-F <features...>] [--all-features] [--no-default-features] [--target <target>] [--color <color>] [<command>] [<args>]

List and diff the public API of Rust library crates between releases and commits.

Options:
  --manifest-path   path to `Cargo.toml`.
  -p, --package     name of package in workspace to list or diff the public API
                    for.
  --omit            omit specified items.
  -F, --features    space or comma separated list of features to activate
  --all-features    activate all available features
  --no-default-features
                    do not activate the `default` feature
  --target          build for the target triple
  --color           when to color the output. By default, `--color=auto` is
                    active. Using just `--color` without an arg is equivalent to
                    `--color=always`.
  --help, help      display usage information

Commands:
  diff              diff the public API against a published version of the
                    crate, or between commits.
